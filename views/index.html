URL Shortener MicroService

Enter a URL to get a shortened link!

Please enter a URL to get started!

How to use the Microservice:

Enter a URL in the input field above and the service will give you a JSON object that has a "short_url" field. Add the value of that field to the end of this domain and it will redirect you to your original URL!

Example Usage:

ancient-ocean-01763.herokuapp.com/api/shorturl/1

will redirect to:

google.com

Sample Request(Python):

import requests

url = "https://ancient-ocean-01763.herokuapp.com/api/shorturl"

payload="{\"url\":\"https://google.com\"}"

headers = { 'Content-Type': 'application/json' }

response = requests.request("POST", url, headers=headers, data=payload)

print(response.text)
